## Dev Settings

### venv

```bash
python -m venv
```
```bash
source venv/bin/activate
```

### install requirements

```bash
pip install -r requirements.txt
```

### customize env

```
DATABASE_URL=sqlite:///database.db
```

### run

```bash
make dev
```
